下面是基于代码整理的客户端功能点清单，可直接作为后续产品形态设计和重构输入。分析范围覆盖 `src`、`su-appcenter`、`su-file`、`su-shell`、`SClaw` 等核心模块。

**0. 客户端壳层**
- Electron 桌面客户端：主窗口、多子窗口、Shell 窗口、传输窗口、输出窗口、图形窗口。
- 全局导航：SClaw、应用、作业、命令行、图形、文件。
- 顶部状态区：上传/下载/跨域/压缩任务入口，任务数量提醒。
- 用户菜单：个人中心、费用中心、我的资源、关于、退出登录。
- 帮助服务：帮助文档、意见反馈、在线客服、业务顾问电话、打开本地日志。
- 更新机制：客户端版本检查、更新包提示、安装更新。
- 权限控制：管理员、普通用户、维护角色、BGP 运营商、资源账户状态、区域维护状态。
- 资源引导：无计算资源提示、资源申请入口、试用/购买/资源配置相关跳转。
- 数据埋点：登录、SClaw、Shell、应用、页面点击等行为上报。

**1. 登录模块**
- 账号登录：用户名/手机号/邮箱 + 密码登录。
- 手机号登录：国家区号选择、短信验证码、60 秒倒计时。
- 图形验证码：按服务端策略展示验证码。
- 密码加密：登录密码 RSA 加密传输。
- 记住密码：本地加密保存账号密码。
- 7 天自动登录：token 登录、本地登录态维护。
- 注册：跳转网页端注册页。
- 找回密码：邮箱找回、手机号找回。
- 重置密码：短信校验后设置新密码，校验密码长度、字符类型和复杂度。
- 登录态管理：accessToken 存储、退出清理、最近登录用户列表维护。
- 登录后初始化：用户信息、SClaw 白名单、区域/集群、公告、版本、License、优惠券等数据加载。

**2. SClaw 模块**
- 助手对话：新建对话、历史会话、流式回复、停止生成、刷新、工具调用状态。
- 消息内容：Markdown 渲染、图片块、工具结果、思考过程开关。
- 文件输入：本地文件选择、拖拽、粘贴、暂存上传、预览、失败重试。
- 技能调用：技能搜索、多技能选择、发送时自动注入技能使用意图。
- 推荐助手：文献追踪、SCNet 智能管家、安全技能审查、网页操控等预设入口。
- 会话历史：按今天、昨天、一周内、两周内、一月内、更早分组。
- 技能管理：技能列表、启用/停用、核心技能标识、缺失依赖、版本、来源路径。
- 技能配置：API Key、环境变量、配置保存、路径复制/打开。
- 消息频道：微信、钉钉、飞书/Lark、企业微信、QQ Bot 等通道配置。
- 频道账号：账号连接状态、二维码/Token 流程、重连、删除、默认账号。
- Agent 管理：Agent 列表、新建、删除、默认 Agent、频道账号绑定。
- 定时任务：任务列表、启停、立即执行、新建、编辑、删除。
- 定时规则：cron 表达式、常用预设、下次运行预览。
- 模型/用量：模型 Provider 配置、Token 使用记录、按模型或时间查看。
- 设置：工作空间目录、模型服务配置、语言、主题、代理、更新、Gateway 控制信息。

**3. 应用模块**
- 我的应用：应用卡片列表、搜索、排序、隐藏/移除、刷新。
- 我的图形：图形类应用入口，支持图形提交和会话创建。
- 我的模板：提交模板保存、读取、删除、最近模板提示。
- 应用中心：应用模板列表、我的应用筛选、我的图形筛选、分类筛选、搜索。
- 商品/已购应用：购买状态、到期状态、安装状态、配置中状态展示。
- 应用订阅：添加到我的应用、移除、申请开通、跳转商城购买。
- 应用资料：用户手册、视频、Demo、Key 等辅助链接。
- 提交方式：模板提交、命令行提交、GUI 图形提交、URL、Docker。
- 作业提交页：动态表单、文件选择、参数输入、资源/队列选择、软件版本选择。
- 智能提交：推荐集群/版本/限制信息、进度、提示信息。
- 图形提交：VNC、DCV、RDP、VM 类型会话创建。
- Linux 桌面：Linux Desktop 会话创建与打开。
- Linux 三维：Linux Desktop 3D/DCV 会话创建与打开。
- Windows 桌面：Windows Desktop/RDP 会话创建与打开。
- 我的产品：已购产品、安装/重装/删除/更新、安装进度、资产下载。
- Notebook 产品：启动、停止、重启、访问地址、状态查询、取消启动。
- 应用安装联动：命令行应用安装状态、软件信息、安装进度弹窗。

**4. 作业列表**
- 当前作业：运行中、排队中、挂起、等待、取消中等状态列表。
- 历史作业：完成、失败、取消等历史记录列表。
- 提交中作业：提交流程中的作业记录入口。
- 筛选查询：状态、标签、类型、应用、队列、区域、时间范围、关键字。
- 表格能力：分页、排序、复制作业 ID、隐藏列、批量选择。
- 作业操作：取消、挂起、释放、批量操作。
- 容器作业入口：SSH、Jupyter、CodeServer、RStudio、容器详情。
- 图形作业入口：noVNC、VNC、DCV 远程桌面。
- 输出查看：标准输出、错误输出、全屏、自动刷新、下载。
- 文件管理：进入工作目录、浏览作业文件、下载文件。
- WinSCP：Windows 客户端下打开文件传输。
- 当前作业详情：基础信息、资源信息、时间限制、运行时长、标签、远程会话。
- 历史作业详情：结束时间、运行时长、转移作业信息、历史文件管理。
- 作业分析：指标分类、节点选择、时间范围、聚合函数、图表分析。
- 操作日志：作业生命周期操作记录。
- 可视化分析：作业性能/结果可视化入口。
- AI 日志辅助：作业输出日志可唤起 AI 辅助分析。

**5. 命令行模块**
- E-Shell：Web Shell/XTerm 终端访问集群。
- 区域选择：多集群选择、默认区域设置、区域维护/停用提示。
- 连接模式：直连模式、中转模式。
- 终端会话：多标签、标签重命名、关闭、切换。
- 分屏布局：横向/纵向分屏。
- 登录节点：节点选择、随机节点模式、当前登录节点展示。
- 文件面板：Shell 左侧嵌入文件管理，支持上传列表联动。
- 指令工具：帮助菜单、调度命令、SC Client 命令、命令粘贴执行。
- 队列信息：区域队列列表、队列资源占用展示。
- Shell 设置：主题、字体、字号、默认区域。
- 密钥管理：SSH Key 信息、生成、重置、下载。
- 本地客户端：客户端下载、检查、日志下载。
- 环境检测：Shell 环境检查、环境修复、恢复环境弹窗。
- 软件信息：应用提交场景下展示软件信息，支持一键粘贴命令。
- AI 辅助：终端内保留 AI Helper 提示控制和命令联动能力。

**6. 图形模块**
- 当前图形：运行中图形会话列表。
- 历史图形：历史图形作业/会话列表。
- 图形类型：VNC、DCV、RDP、VM。
- 当前图形筛选：区域、应用、版本。
- 历史图形筛选：图形名称、作业 ID、类型、区域、应用、时间范围。
- 会话状态：运行、排队、保持、启动中、停止中、已停止、创建失败、销毁中。
- 图形操作：打开、删除/销毁、取消排队、查看日志。
- VM 操作：开机、关机、创建进度轮询。
- 时间管理：DCV/VM 续期、剩余时间展示。
- 截图预览：VNC/DCV 会话截图展示。
- 客户端支持：DCV 客户端下载入口。
- 作业联动：历史图形可跳转当前/历史作业详情。

**7. 文件模块**
- 文件列表：按区域/路径浏览文件和目录。
- 文件树：目录树、路径面包屑、路径切换。
- 基础操作：新建目录、新建文件、重命名、删除。
- 复制移动：文件复制、移动、冲突处理、跨路径操作。
- 上传文件：普通上传、分片上传、上传队列。
- 下载文件：下载检查、下载队列、客户端下载联动。
- 文件预览：文本、图片、Office 文件预览。
- 文本编辑：读取、编辑、保存文本文件。
- 文件搜索：按路径和关键字搜索。
- 文件分享：开启分享、关闭分享、分享码、分享记录。
- 异步任务：删除、复制、移动、目录大小统计等任务列表与取消。
- 跨域传输：发起跨区域传输、暂停、继续、取消、删除记录。
- 压缩解压：提交压缩/解压任务、任务列表、取消任务。
- 高速传输：Raysync/Fasttrans 能力、Linux 命令弹窗、同步状态。
- 操作记录：传输记录、跨域记录、压缩记录、异步任务记录。

**8. 个人中心**
- 用户资料展示：姓名、用户名、行业、领域、单位、部门、职务。
- 地区信息：国家/地区、省市、地址。
- 时间信息：创建时间、修改时间。
- 资料编辑：跳转网页端个人资料编辑页。
- 安全等级：根据密码强度、手机、邮箱、微信绑定计算安全等级。
- 密码状态：密码强度展示。
- 手机安全：手机号绑定/验证状态展示。
- 邮箱安全：邮箱绑定/验证状态展示。
- 微信绑定：微信绑定状态展示。
- 实名认证：认证状态展示，支持跳转认证页面。
- 资源提示：个人中心顶部展示计算资源申请提示。
