## 5. 重构风险等级

| 风险等级 | 功能 | 风险说明 | 建议 |
|---|---|---|---|
| 极高 | 初始化、权限、资源状态 | 入口错显会导致全局不可用或越权 | 先建立全局状态字典和测试矩阵 |
| 极高 | 文件传输和任务中心 | 数据任务失败代价高 | 统一任务模型，保留关闭保护 |
| 极高 | 应用提交和作业管理 | 计算主链路直接影响客户价值 | 端到端验证提交、列表、详情、日志 |
| 高 | 图形会话 | 类型多，窗口和作业联动复杂 | 建立统一会话模型和类型适配 |
| 高 | SClaw | 宿主与子应用边界复杂 | 固化白名单、服务、埋点和消息桥接 |
| 中 | 个人中心、资源外链 | 控制台承载边界清晰即可 | 标注文档外链能力 |
| 中 | 帮助、反馈、日志、更新 | 影响支持和维护 | 保留入口和异常兜底 |
| 低 | 公共页脚、免责声明 | 静态展示为主 | 确认触发和合规要求 |

## 6. AI 辅助适配优先级

| 优先级 | 场景 | 适配原因 |
|---|---|---|
| P0 | 作业失败日志分析 | 用户痛点强，已有作业与 SClaw 联动 |
| P0 | 文件/跨域/上传失败解释 | 错误类型多，AI 可提供下一步 |
| P1 | 资源不足与权限阻断解释 | 前置条件复杂，用户需要路径指引 |
| P1 | 图形创建失败和连接异常 | 多状态多依赖，适合诊断 |
| P1 | Shell 连接异常 | 网络、账号、节点问题可被结构化解释 |
| P2 | 新手引导个性化 | 可根据用户角色推荐路径 |
| P2 | 帮助文档推荐和反馈生成 | 可降低客服压力 |
