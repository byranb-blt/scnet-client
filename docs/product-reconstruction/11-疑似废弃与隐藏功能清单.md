# 疑似废弃 / 隐藏功能清单

## 1. 疑似废弃或保留待确认功能

| 功能 | 位置 | 判断依据 | 建议 |
|---|---|---|---|
| DeepSeek 页面 | `src/router/index.js` `/home/deepseek` | 路由存在，主导航入口未在本轮确认 | 确认是否归入 SClaw 或历史页面 |
| 免责声明弹窗 | `src/Layout/components/Disclaimer.vue` | 组件存在，触发链路待确认 | 法务确认展示规则 |
| 旧资源申请跳转 | `src/views/applySource/index.vue` | 跳转旧控制台 `/ac/console3/index.html#/ware/apply-cluster` | 确认是否保留旧入口 |
| Shell 工作流入口 | `src/views/shell/index.vue` | 入口代码保留，当前配置关闭 | 确认产品计划 |
| ICA/Frame 图形分支 | `CreateGraph.vue` | 代码中有历史分支和改造痕迹 | 确认线上应用配置 |
| 当前图形表格视图 | `GraphList.vue` | 表格视图代码存在，切换入口疑似隐藏 | 确认是否保留 |
| 旧版上传/下载独立窗口 | `main/window/upload.js`、`main/window/download.js`、`main/window/transfer.js` | 旧窗口和统一传输窗口并存 | 明确产品边界 |
| SClaw Agents/Models 用户入口 | `submodules/SClaw/src/App.tsx` | 路由存在，侧边栏入口需确认 | 明确开放范围 |
| 个人中心申请更多资源提示 | `userCenter/index.vue` | `sourseFlag` 初始关闭，方法链路待确认 | 确认是否合并到资源中心 |
| 开发者更新调试 | `DeveloperPanel.vue`、`main/dev-panel/index.js` | 面向内部调试，普通用户无入口 | 单列内部诊断能力 |

## 2. 隐藏但已确认存在的产品能力

| 功能 | 位置 | 判断依据 | 建议 |
|---|---|---|---|
| 主窗口关闭到托盘 | `main/index.js` | 主窗口关闭行为和托盘恢复逻辑存在 | 作为客户端生命周期能力保留 |
| 退出保护 | `main/index.js`、`main/window/*` | 子窗口或任务运行中退出会提示 | 作为数据安全能力保留 |
| 打开本地日志目录 | `Layout/index.vue`、`main/log/core.js` | 客服下拉可打开日志 | 作为排障入口保留 |
| 开发者日志查看器 | `dev-panel/LogViewer.vue` | 开发者入口可查看 app/sclaw/gateway 日志 | 归为内部支持能力 |
| 公告永久关闭 | `Announcement.vue` | 公告支持关闭和永久关闭 | 需要补充恢复策略 |
| 多模块新手引导 | `newGuide/guide.vue` | 覆盖 SClaw、应用、作业、Shell、图形、文件 | 统一引导体系 |
| 工具下载 | `ToolsDownload.vue` | FastTrans 或 Receiver 缺失时提示下载 | 归入本地工具依赖能力 |
| 下载路径选择 | `selectPath/index.vue` | 本地下载目录选择 | 保留为客户端下载体验 |
| SClaw 白名单 | `sclaw-whitelist.js` | 入口由白名单控制 | 产品需确定未通过提示策略 |
| SClaw 埋点桥接 | `Layout/index.vue`、`sensor-bridge.ts` | 子应用事件转发到宿主神策 | 保留统计链路 |
| VM 停止/删除关闭窗口 | `CreateGraph/Vm.vue` | VM 状态操作会联动窗口 | 保留窗口生命周期联动 |
| 文件异步任务通知 | `async-task/info.js` | 复制/移动/删除任务有通知入口 | 归入任务中心 |
| Linux 快传命令 | `LinuxCommandModal.vue` | 支持服务器端快传命令 | 归入文件高级传输 |
| DCV 访问文件下载 | `CreateGraph/Dcv.vue` | DCV 会话可下载访问文件 | 归入图形辅助能力 |

## 3. 需要产品决策的历史/边缘能力

| 功能 | 当前建议 | 原因 |
|---|---|---|
| DeepSeek 页面 | 合并到 SClaw 或归档 | 智能助手已有一级入口 |
| 旧资源申请页 | 合并到新资源中心 | 资源入口应统一 |
| 旧上传/下载窗口 | 合并到统一传输窗口 | 用户只需要一个任务中心 |
| Shell 工作流 | 明确计划后再保留入口 | 当前配置关闭，用户价值未闭环 |
| ICA/Frame | 确认线上应用后保留 | 图形类型过多会增加理解成本 |
| 开发者面板 | 保留内部入口 | 对用户主流程无直接价值，但排障重要 |

## 4. 重构保护清单

| 能力 | 保护原因 |
|---|---|
| 退出保护 | 防止运行中上传、下载、子窗口导致数据丢失 |
| 打开日志 | 客服和研发排障关键入口 |
| 工具下载 | 快传和远程桌面依赖本地工具 |
| 公告和新手引导 | 平台通知与新用户学习入口 |
| SClaw 白名单 | 智能助手准入控制 |
| BGP 入口隐藏 | 明确角色差异 |
| 区域维护和账号停用空态 | 资源异常时的核心用户解释 |
