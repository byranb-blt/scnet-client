# 权限 / 条件字典

## 1. 全局权限与条件

| 条件 | 影响范围 | 用户表现 | 建议说明 |
|---|---|---|---|
| 登录态有效 | 全部登录后功能 | 可进入主框架和业务页面 | 所有业务能力的第一前置条件 |
| 登录过期 | 全部业务请求 | 提示重新登录并回登录页 | 需要保留清晰的会话过期提示 |
| 用户信息加载成功 | 导航、个人中心、角色、埋点 | 入口按用户状态展示 | 初始化失败应有重试或回登录策略 |
| 团队管理员 | 费用、资源购买、资源申请、应用无资源引导 | 显示购买/管理员入口 | 资源相关能力应明确管理员和成员差异 |
| 普通成员 | 无资源引导、资源购买 | 显示联系团队管理员 | 成员自助购买能力受限 |
| BGP 操作员 | Shell、下载、跨域、费用、文件菜单 | 入口隐藏或功能减少 | 重构时需完整迁移可见性规则 |
| SClaw 白名单 | SClaw 入口、AI 日志分析 | 入口显示或隐藏 | 白名单失败时是否提示用户需确认 |
| 开发者入口 | 开发者面板、日志查看、更新调试 | 普通用户无入口 | 内部能力应与用户功能隔离 |

## 2. 资源与区域条件

| 条件 | 影响范围 | 用户表现 | 建议说明 |
|---|---|---|---|
| 有可用资源 | 文件、作业、Shell、图形、应用提交 | 模块可正常使用 | 登录后初始化应准确判断 |
| 无可用资源 | 文件、当前作业、应用提交、图形 | 开通计算资源弹窗或应用内引导 | 引导应包含购买、申请、刷新和联系管理员 |
| 区域 active | 区域内业务 | 可选择和操作 | 正常状态 |
| 区域 underMaintenance | 文件、作业、Shell、图形、跨域、后处理 | 维护空态、按钮禁用、操作阻断 | 强阻断状态需统一文案 |
| 区域 inactive | 区域相关功能 | 停用空态 | 提供客服或购买其他区域资源入口 |
| 区域异常 | 区域相关功能 | 异常提示或入口禁用 | 应提供重试和切换区域 |
| 资源账号正常 | 作业提交、Shell、文件、图形 | 可执行资源操作 | 核心前置 |
| 资源账号 Disable | 作业、文件、Shell、图形、后处理 | 账号停用提示 | 应明确联系对象 |
| 存储空间充足 | 上传、跨域、压缩、作业提交 | 可提交任务 | 存储不足应引导资源或文件清理 |
| 多区域资源 | 跨域传输 | 可选择源/目标区域 | 缺少多区域资源时引导申请 |
| 队列/预算可用 | 作业、图形独占会话 | 可提交 | 异常时应解释队列或预算问题 |
| 会话数量未超限 | VNC/DCV 图形会话 | 可创建 | 超限时应引导释放已有会话 |

## 3. 应用与软件条件

| 条件 | 影响范围 | 用户表现 | 建议说明 |
|---|---|---|---|
| 应用已购买 | 应用启动、提交、图形创建 | 按钮可用 | 未购买进入购买或开通引导 |
| 应用未购买 | 应用详情、提交 | 按钮禁用或引导购买 | 应区分未购买和无资源 |
| 应用到期 | 我的应用、提交 | 到期提示 | 应提供续费或购买入口 |
| 软件已安装 | 作业提交、图形创建 | 版本可选择 | 未安装应提示安装或联系管理员 |
| 软件版本可用 | 提交和图形启动 | 可选择版本 | 无版本时应给出明确文案 |
| 应用支持 GUI | 图形提交、图形应用中心 | 显示图形入口 | 应用交付方式决定图形流程 |
| 应用支持模板 | 模板提交 | 显示模板参数 | 模板缺失时需兜底 |
| 应用支持当前区域 | 应用提交资源选择 | 区域可选 | 已有资源不支持应用时展示支持区域 |

## 4. 文件与任务条件

| 条件 | 影响范围 | 用户表现 | 建议说明 |
|---|---|---|---|
| 文件读权限 | 下载、预览、跨域源、压缩源 | 操作可用 | 权限不足应禁用或提示 |
| 文件写权限 | 上传、新建、删除、重命名、移动、解压目标 | 操作可用 | 写权限是文件操作核心条件 |
| 目录执行权限 | 进入目录、路径跳转 | 可进入目录 | 不可进入时提示权限不足 |
| Root 所属文件 | 删除、移动、编辑、重命名 | 普通用户操作受限 | 需要保护系统和共享数据 |
| 文件名合法 | 上传、新建、重命名 | 可提交 | 特殊字符应提前提示 |
| 文件数量限制 | 上传文件夹、批量下载 | 超限时阻断 | 上传文件夹超限建议快传或压缩 |
| 文件大小限制 | 预览、快传、图形后处理 | 超限时阻断 | 提示替代路径 |
| 快传客户端已安装 | 快速上传/下载 | 快传可用 | 缺失时弹出工具下载 |
| 本地下载目录可用 | 客户端下载 | 可保存和打开文件 | 目录不可用时应提示重新选择 |
| 运行中任务存在 | 关闭窗口/退出应用 | 弹出关闭保护 | 防止数据任务中断 |

## 5. SClaw 与 AI 条件

| 条件 | 影响范围 | 用户表现 | 建议说明 |
|---|---|---|---|
| 白名单通过 | SClaw 入口 | 左侧展示 SClaw | 智能助手入口前置 |
| Gateway 运行中 | 对话、技能、定时任务 | 可正常使用 | 服务异常应提供重启/恢复 |
| 模型 Provider 已配置 | 对话、Agent、技能 | 可调用模型 | 配置缺失时引导设置 |
| API Key 有效 | 模型调用 | 对话成功 | 验证失败时提示设置 |
| 额度充足 | 对话、技能执行 | 正常返回 | 额度不足时给出跳转链接 |
| 技能已安装/启用 | 技能选择 | 可使用技能 | 未安装时引导安装 |
| 频道已绑定 | 定时任务通知 | 可推送消息 | 绑定失败时给出二维码或配置提示 |

## 6. 控制台与外链条件

| 条件 | 影响范围 | 用户表现 | 建议说明 |
|---|---|---|---|
| 控制台地址可用 | 个人资料、费用、资源、反馈 | 可打开外部页面 | 外链失败需兜底 |
| 登录 token 可用 | 反馈、控制台免登录跳转 | 自动进入目标页 | token 失效需回登录 |
| 资源订单存在 | 无资源弹窗 | 查看订单、继续购买 | 订单状态应清晰 |
| 资源申请单存在 | 无资源弹窗 | 已提交申请提示 | 需要展示当前状态 |
| 客服服务可用 | 在线咨询 | 打开客服窗口 | 客服失败需提示和日志入口 |
| 帮助文档地址可用 | 帮助文档 | 打开文档 | 文档缺失需兜底 |

## 7. 更新与本地客户端条件

| 条件 | 影响范围 | 用户表现 | 建议说明 |
|---|---|---|---|
| 客户端已打包 | 自动更新 | 可检查和安装更新 | 开发环境跳过检查 |
| 版本服务可用 | 强弱更新提示 | 正常展示版本信息 | 服务异常需记录日志 |
| 弱更新标记 | 顶部提示、关于红点 | 可升级或关闭 | 用户可延后 |
| 强更新标记 | 全局强更新弹窗 | 只能立即升级 | 阻断业务使用 |
| 更新包下载完成 | 安装升级 | 显示升级入口 | 安装失败需兜底 |
| 本地日志目录可打开 | 打开日志 | 打开文件夹 | 失败时应提示路径或重试 |
| 系统托盘可用 | 关闭到托盘 | 窗口隐藏、托盘存在 | 平台差异需保留 |
