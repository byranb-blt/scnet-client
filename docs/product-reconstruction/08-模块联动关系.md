# 模块联动关系

## 1. 全局联动矩阵

| 来源模块 | 目标模块 | 联动方式 | 产品影响 |
|---|---|---|---|
| 登录 | 初始化 | 登录成功后触发用户、区域、资源、公告、版本加载 | 决定后续功能可用性 |
| 初始化 | 壳层 | 根据角色、资源、白名单渲染导航和弹窗 | 影响入口可见性 |
| 初始化 | 权限 | 写入角色、区域、账号和资源状态 | 全局拦截和提示来源 |
| 壳层 | SClaw | 左侧入口、webview 承载、关闭/恢复层 | 智能助手作为主模块展示 |
| 壳层 | 任务中心 | 顶部任务入口和独立窗口 | 长耗时任务可见 |
| 壳层 | 帮助/更新 | 顶部帮助、反馈、客服、版本提示 | 支持和维护能力 |
| 应用 | 作业 | 应用提交生成作业 | 计算任务主链路 |
| 应用 | 图形 | 图形应用进入 VNC/DCV/RDP/VM | GUI 主链路 |
| 应用 | 文件 | 提交参数和输入输出路径复用文件选择器 | 形成数据输入链路 |
| 应用 | 资源 | 根据应用支持区域和资源状态显示可用性 | 影响提交成功率 |
| 作业 | 文件 | 作业详情进入工作目录、输出和结果文件 | 结果查看和下载 |
| 作业 | 图形 | GUI 作业关联图形会话 | 交互式作业链路 |
| 作业 | SClaw | 作业日志唤起 AI 分析 | 智能排障 |
| 作业 | 任务中心 | 压缩/解压等作业化任务进入任务视图 | 文件任务状态可追踪 |
| 文件 | 任务中心 | 上传、下载、跨域、压缩、异步任务汇入任务中心 | 统一长耗时任务 |
| 文件 | Shell | 右键 Shell 打开目录，Shell 内嵌文件面板 | 命令与文件联动 |
| 文件 | 图形 | 文件后处理入口进入 VNC/DCV 应用 | 特定后处理场景 |
| 文件 | 资源 | 区域、账号、权限、存储影响文件操作 | 数据操作安全 |
| Shell | 文件 | 终端侧栏浏览文件、rz 上传、下载任务 | 命令行文件处理 |
| Shell | 任务中心 | 上传下载进入全局任务 | 任务状态统一 |
| 图形 | 壳层 | VNC/DCV/RDP/VM 独立窗口 | 远程桌面体验 |
| 图形 | 作业 | 独占图形会话通过 GUI 作业排队运行 | 图形和调度结合 |
| 图形 | 资源 | 会话数量、节点、存储、账号决定创建结果 | 影响可用性 |
| 个人中心 | 资源 | 实名和账号状态影响资源相关能力 | 账号可信度 |
| 资源 | 全模块 | 无资源、区域维护、账号停用阻断业务 | 全局强依赖 |
| 帮助/日志 | 全模块 | 异常后打开文档、客服、日志 | 问题闭环 |
| 更新 | 壳层 | 强更新弹窗阻断，弱更新提示 | 版本维护 |
| 数据埋点 | 全模块 | 采集登录、SClaw、应用、作业、文件、Shell 行为 | 产品分析 |
| 其他 | 壳层/任务 | 公告、引导、托盘、退出保护 | 横向体验和安全 |

## 2. 关键联动路径

### 2.1 应用到作业

| 节点 | 说明 |
|---|---|
| 起点 | 应用中心或我的应用 |
| 中间能力 | 应用详情、模板提交、文件选择、资源选择、参数校验 |
| 结果 | 作业提交成功后进入当前作业或提交中作业 |
| 依赖 | 应用购买、软件版本、资源账号、区域、存储、队列 |
| 异常 | 无资源、软件未开通、应用到期、提交失败 |

### 2.2 作业到文件、图形、日志、AI

| 节点 | 说明 |
|---|---|
| 起点 | 作业列表或作业详情 |
| 文件联动 | 查看工作目录、输出文件、结果下载 |
| 图形联动 | GUI 作业可关联图形会话 |
| 日志联动 | 查看 stdout、stderr、运行日志、全量输出 |
| AI 联动 | SClaw 日志分析和选中文本辅助 |
| 异常 | 无日志、文件权限不足、SClaw 不可用 |

### 2.3 文件到任务中心、Shell、图形

| 节点 | 说明 |
|---|---|
| 起点 | 文件管理页或文件选择器 |
| 任务中心 | 上传、下载、跨域、压缩、异步复制移动删除 |
| Shell | 右键进入 Shell 并切换到当前目录 |
| 图形 | 右键后处理入口打开图形处理流程 |
| 异常 | 权限不足、账号停用、区域维护、存储不足 |

### 2.4 图形会话到应用、作业、窗口

| 节点 | 说明 |
|---|---|
| 起点 | 图形应用中心、应用图形提交、作业详情 |
| 应用联动 | 应用交付方式决定 VNC/DCV/RDP/VM |
| 作业联动 | 独占图形会话通过 GUI 作业排队运行 |
| 窗口联动 | 远程桌面通过 VNC/VM 独立窗口承载 |
| 异常 | 会话超限、连接未就绪、创建失败 |

### 2.5 Shell 到文件和软件环境

| 节点 | 说明 |
|---|---|
| 起点 | 左侧命令行或文件右键 |
| 文件联动 | 文件侧栏、路径拖入、上传下载任务 |
| 软件联动 | 应用中心 Shell、软件信息复制/粘贴 |
| 资源联动 | 区域、集群、登录节点、账号状态 |
| 异常 | 连接失败、节点异常、账号停用 |

### 2.6 SClaw 到宿主和业务模块

| 节点 | 说明 |
|---|---|
| 起点 | 左侧 SClaw 或作业日志 AI 分析 |
| 宿主联动 | webview 承载、状态同步、埋点桥接 |
| 作业联动 | 日志分析、输出辅助 |
| 配置联动 | 模型、技能、频道、定时任务 |
| 异常 | 白名单、Gateway、模型配置、额度 |

## 3. 重构优先级联动

| 优先级 | 联动关系 | 原因 |
|---|---|---|
| P0 | 登录 → 初始化 → 权限 → 壳层 | 所有功能前置链路 |
| P0 | 应用 → 作业 → 文件 | 计算主价值链路 |
| P0 | 文件 → 任务中心 | 数据流转核心链路 |
| P0 | 图形 → 独立窗口 → 资源 | GUI 核心链路 |
| P1 | 作业 → SClaw | AI 排障重要能力 |
| P1 | Shell → 文件 | 高频效率能力 |
| P1 | 资源 → 应用/作业/文件/图形/Shell | 全局阻断条件 |
| P2 | 帮助/客服/日志 → 全模块 | 支持闭环 |
| P2 | 更新 → 壳层 | 版本维护 |
| P3 | 埋点 → 全模块 | 产品分析 |
